간수치 상승·지방간 결과 안내

간수치(AST/ALT/GGT) 상승은 흔하지만 원인은 다양합니다. 지방간, 음주, 약·영양제, 바이러스간염, 근육 손상 등을 함께 확인하고 위험도에 따라 재검과 추가 검사를 정합니다.

흔한 원인과 확인할 검사

흔한 원인
  • 지방간 — 복부비만·당뇨·고중성지방과 동반
  • 음주 — GGT 상승, AST가 ALT보다 높을 수 있음
  • 약·영양제 — 진통제, 한약, 보충제 포함
  • 운동·근육 손상 — AST/CK 동반 상승 가능
추가 확인
  • 간염 검사 — B형·C형 간염 여부
  • 복부초음파 — 지방간·담석·간 종괴 확인
  • 대사 위험 평가 — 혈당, 지질, 혈압, 허리둘레
  • 섬유화 평가 — FIB-4, 필요 시 탄성도 검사

생활 관리의 우선순위

먼저 줄일 것
  • 술 — 재검 전 2~4주 금주하면 원인 판단에 도움
  • 당 음료·과자 — 과당은 지방간 악화와 연관
  • 야식·폭식 — 체중보다 복부지방 감소가 중요
  • 불필요한 보충제 — 성분 불명 제품은 중단 상담
늘릴 것
  • 주 150분 유산소 — 빠르게 걷기부터 시작
  • 주 2회 근력운동 — 인슐린 저항성 개선
  • 단백질·채소 중심 식사 — 흰쌀·면·빵 양 줄이기
  • 3개월 재평가 — 간수치·체중·허리둘레 비교
빨리 진료가 필요한 신호
황달, 콜라색 소변, 심한 우상복부 통증, 발열, 멍·코피 증가, 복부 팽만, 원인 모를 체중 감소가 있으면 단순 지방간으로 넘기지 말고 빨리 진료를 보세요. 간수치가 정상이어도 지방간·섬유화가 있을 수 있어 위험요인에 따라 추적이 필요합니다.
